# Задача вибору сценаріїв розширення потужностей у транспортній моделі з бюджетним обмеженням

Курсова робота з дисципліни **«Дослідження операцій в інформаційно-управляючих системах»**

**Група:** ІС-32  
**Автори:** Кіселар Владислав, Ковалюк Валерія, Топка Тіна, Черепня Юрій

---

## Опис

Програма розв'язує задачу транспортного типу з вибором сценаріїв розширення потужностей постачальників за умови бюджетного обмеження. Реалізовано два алгоритми — жадібний та генетичний — з можливістю порівняння їх результатів.

---

## Встановлення залежностей

```bash
pip install -r requirements.txt
```

---

## Запуск

```bash
python main.py start
```

Програма запускає інтерактивне меню, через яке можна:
- ввести або згенерувати дані задачі
- розв'язати задачу обома алгоритмами
- переглянути результати
- провести обчислювальні експерименти

Результати експериментів зберігаються у папці `output/`.
